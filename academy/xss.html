💥 What is XSS?

Cross-site scripting (XSS) is a client-side code injection vulnerability...

🧪 Types of XSS

  • Stored XSS: Payload stored in DB
  • Reflected XSS: Payload reflected from request
  • DOM-based XSS: Exploiting browser-side JS

📌 Example Payload

<script>alert('XSS');</script>

🛡️ Prevention

  • Use proper output encoding
  • Sanitize inputs
  • Use CSP (Content Security Policy)

💡 Tip

Try injecting payloads in input fields, URL parameters, or hidden fields.
