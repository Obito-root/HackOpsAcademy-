🔌 Chapter 5: Web Service Ports (HTTP/HTTPS)

Different services run on specific ports. Recognizing them is crucial in reconnaissance and scanning stages of ethical hacking.

🗂️ Common Web Ports

Port Protocol Service Use
80 TCP HTTP Standard web browsing
443 TCP HTTPS Secure web browsing
8080 TCP HTTP-Alt / Proxy Used for proxies

💡 Tip

Use nmap -p 80,443,8080 [target] to scan these common web service ports.
