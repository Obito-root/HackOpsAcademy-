🧰 Chapter 2: Setting Up Basic Tools

Hackers use a wide variety of tools categorized by their function in different stages of hacking. These categories include:

📊 1. Information Gathering

  • Tools that collect data about the target (e.g., Whois, Nmap, Recon-ng)
  • Useful for footprinting and enumeration

🔍 2. Vulnerability Analysis

  • Tools to detect known vulnerabilities (e.g., Nessus, OpenVAS, Nikto)
  • Helps in identifying potential weaknesses

💣 3. Exploitation

  • Tools to exploit vulnerabilities (e.g., Metasploit, SQLmap, BeEF)
  • Used to gain unauthorized access or run payloads

💡 Tip

Set up a Linux-based environment (like Kali Linux) and install these tools to get hands-on experience safely in a lab setup.
