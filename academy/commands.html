📘 Chapter 3: How Linux Commands Work

Linux commands follow a structured syntax to perform powerful actions. Understanding this structure is essential for every ethical hacker.

⚙️ Basic Structure

[command] [subcommand] [argument] [option/flag]

Example:

apt install nmap -y

🔍 Breakdown

  • apt → Main command (Advanced Package Tool)
  • install → Subcommand (Action to perform)
  • nmap → Argument (Package name)
  • -y → Option/Flag (Auto-confirmation)

💡 Tip

Always read the --help section of a command to explore all available flags and options. For example, try: nmap --help
